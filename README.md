# Multiple Linear Regression from Scratch

This project implements Multiple Linear Regression from scratch using NumPy on the California Housing dataset.  
The project includes Gradient Descent optimization, Stochastic Gradient Descent (SGD), L1 (Lasso) and L2 (Ridge) Regularization, and Multiple Linear Regression using the Normal Equation method.

## Features
- Multiple Linear Regression from scratch
- Gradient Descent Implementation
- Stochastic Gradient Descent (SGD)
- Normal Equation Method
- L1 Regularization (Lasso)
- L2 Regularization (Ridge)
- Feature Standardization
- Cost Function Visualization
- Performance Evaluation using MAE, MSE, and RMSE

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
California Housing Dataset from Scikit-learn.

## Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

## Learning Outcome
This project demonstrates the mathematical implementation of Multiple Linear Regression and optimization techniques used in Machine Learning.
